결석관리
     -학생-
	1)결석신청
	       결석 관리 탭을 누르면 ->자신의 결석 신청진행중인 게시판이 나온다 -> 게시판 밑에 결석 신청버튼을 누른다 ->
	       결석신청 양식 페이지로 이동 -> 양식에 맞춰 작성(RECIPIENT_ID, COURSECODE, REASON, START_DATE, ABSENT_DAYS, FILENAME, NEWFILENAME)
		 -> 양식 밑에 있는 신청하기 버튼을 누른다(취소하기 버튼 누르면 게시판화면)-> 작성한 신청성의 seq와 세션에 담겨있는 id 를 찾아서 신청상태테이블(stm='N')
		 -> 결석 신청진행중인 게시판화면이동
	2)결석 신청 내역조회
		결석 관리 탭 클릭->자신의 결석 신청진행중인 게시판화면(APP_DATE, COURSENAME, STM,RECIPIENT_ID)
		게시판 상단에 진행중,승인완료,미승인 셀렉트 박스로 각각의 상태별로 모아 볼수 있다.
		신청일을 클릭하면 상세조회가능 -> 신청한 양식나옴/미승인인 신청서 상세 조회시 신청한 양식+반려사유
		승인 완료 신청서 조회시 신청한 양식+담당자의 사인 이미지
     -강사,관리자-
	1)결석 신청 내역조회
		결석 관리 탭을 누르면 처리해야할 신청서게시판 화면(APP_DATE, COURSENAME, STM, NAME)
		게시판 상단에 진행중,승인완료,미승인 셀렉트 박스로 각각의 상태별로 모아 볼수 있다.
		신청일을 클릭하면 상세조회가능 -> 미승인인 신청서 상세 조회시 신청한 양식+반려사유
		승인 완료 신청서 조회시 신청한 양식+담당자의 사인 이미지
	2)신청서 수리
		신청일 클릭시 상세 조회 화면에서 하단에 승인/미승인/ 버튼으로 수리가능
		승인시 상태테이블 상태 변경(승인=Y)
		미승인시 상태 테이블 상태 변경(미승인=R) , 미승인 버튼 클릭시 미승인 사유작성할 수 있는 TEXT AREA 띄어줌(Ajax)->입력후 완료버튼 누르면 해당 신청서의
		seq를 받아와서 반려테이블에 저장
	3)이미지 등록 
		결석 관리 탭을 누르면 게시판 하단에 이미지 등록 버튼 클릭->파일 업로드(이미지 테이블에 저장)
